Constructor

new Game()

Gameコンポーネントを構築する。 Gameコンポーネントのプロパティに以下のプロパティを設定する。
  • history< - Bordコンポーネントに配置された3x3のSquareコンポーネントの 表示内容の履歴の初期値として、3x3のSquareすべてにnullを設定する。
  • stepNumber - 対局のカウンタとして0を設定する。
  • xIsNext - true を設定する。
