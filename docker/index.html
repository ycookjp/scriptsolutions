Shell scripts for building image

fedora38-container-base.sh creating container base for Fedora
fc38_desktop.sh creating desktop environment
fc38_eclipse.sh creating Eclipse IDE with desktop environment
fc38_httpd-proxy.sh creating proxy service working on Apache HTTP Server
fc38_httpd-svn.sh creating SVN service working on Apache HTTP Server

Dockerfiles

Fedora based images
desktop_fc38_Dockerfile creating desktop environment
eclipse_fc38_Dockerfile creating Eclipse IDE with desktop environment
httpd-git_fc38_Dockerfile creating git repository service working on Apache httpd
httpd-proxy_fc38_Dockerfile creating proxy service working on Apache httpd
httpd-proxy-mdns_fc38_Dockerfile creating proxy service with mDNS working on Apache httpd
httpd-svn_fc38_Dockerfile creating SVN repository service working on Apache httpd
Note

Before building docker image, you must create and import "fedora-container-base" image by executing fedora31-container-base.sh .

Or download Dockerfile from Fedora Official Image page and build image.

# wget https://github.com/fedora-cloud/docker-brew-fedora/raw/078f33bbbab02b9f1be80b1b48605c376e82fb18/x86_64/Dockerfile
# wget https://github.com/fedora-cloud/docker-brew-fedora/raw/078f33bbbab02b9f1be80b1b48605c376e82fb18/x86_64/fedora-30-updates-candidate-x86_64-20190609.tar.xz
# docker build -t fedora31-container-base .
CentOS-7 baseed images
centos7-httpd-git_Dockerfile creating Git repository service working on Apache httpd
Note

To building image at closed network, download Dockerfile and official docker image for CentOS 7 from Docker registry page, and build image as following.

# docker build -t centos:7 .
